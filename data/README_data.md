# Data directory

Place the following file here before running any script:

| File | Description |
|------|-------------|
| `sc_mean.csv` | 81 physicochemical descriptors + critical_temp + material (n = 15,542) |

## Download

The file is available at:
https://github.com/matasovav/DATA_SC/blob/master/sc_mean.csv

Download sc_mean.csv and place it at data/sc_mean.csv before running any script.
